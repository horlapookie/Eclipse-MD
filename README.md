🧠 Deployment Notes

Railway

"railway.yaml" is included for easy deployment.

Make sure to set your environment variables in Railway dashboard.

Replit

- Set "BOT_SESSION_DATA" in Secrets
- Run the Eclipse MD Bot workflow
- If "auth_info" folder exists the bot may prioritize it.
