### Bot Configuration
```bash
BOT_PREFIX=.                 # Command prefix
BOT_NUMBER=234*****          # owner number 
BOT_SESSION_DATA=<session>   # WhatsApp session ID
```


## 🚀 Quick Start

### Deployment
This bot can be deployed on Replit, Railway, or any Docker-enabled platform. 

#### Railway Deployment
A `railway.yaml` is included for easy deployment. Ensure you set the required environment variables in your Railway project settings.

#### Replit Setup
- Set `BOT_SESSION_DATA` in Secrets.
- Run the "Eclipse MD Bot" workflow.
- If you have an existing `auth_info` folder, the bot will prioritize it even if `SESSION-ID` is missing.
